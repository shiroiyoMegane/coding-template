.l-colSet01({col:カラム数, margin:間隔})

	+MQ_PC(SP_RES_WID01)

	+MQ_SP(SP_RES_WID01)
		l-colSet01({col:1, margin:SP_MARGIN})

  • m-bgTitle01

コンテンツの前に要素を入れる必要がある場合、
before:trueにして、 mixin beforeGroup()を宣言することで、 要素の挿入ができる。
コンテンツの後ろ、タイトルの前後についても
以下のオプションをtrueにすることで、挿入可能。

小タイトル、中タイトル、大タイトルについても必要であれば、
smallTitle, title, largeTitleを使用することで挿入可能。
(デフォルトで使用するのは title) 

+c-bgTitle01({
	tag:          "div",                //親タグ名
	baseName:     "c-bgTitle01",   //ベースclass名
	title:        "",                   // タイトル
	smallTitle:   "",                   // 中タイトル
	largeTitle:   "",                   // 大タイトル
	className:    "",                   //追加class名
	before:       false,                //コンテンツの前に入る要素
	after:        false,                //コンテンツの後ろに入る要素
	titleBefore:  false,                //タイトルの前に入る要素
	titleAfter:   false                 //タイトルの後ろに入る要素
})

【シンプル】

+m-bgTitle01({tag:"h3", title:"背景タイトル01"})

【全要素表示】

+m-bgTitle01({tag:"h3", largeTitle:"大タイトル", title:"背景タイトル01（全要素表示）", smallTitle:"少タイトル", className:"is-center",before:true, after:true, titleBefore:true, titleAfter:true})
	mixin beforeGroup()
		// コンテンツの前に入る要素
	mixin titleBefore()
		// タイトルの前に入る要素
	mixin titleAfter()
		// タイトルの後に入る要素
	mixin afterGroup()
		// コンテンツの後に入る要素
		p afterテキスト
.m-bgTitle01
	.group.is-primary
		h3.com-title
			span.is-title tg.title
<div class="m-bgTitle01">
	<div class="group is-primary">
		<h3 class="com-title"><span class="is-title">背景タイトル01</span></h3>
	</div>
</div>

あるところは岨づたいに行く崖の道であり、あるところは数十間の深さに臨む木曾川の岸であり、あるところは山の尾をめぐる谷の入り口である。一筋の街道はこの深い森林地帯を貫いていた。
東ざかいの桜沢から、西の十曲峠まで、木曾十一宿はこの街道に添うて、二十二里余にわたる長い谿谷の間に散在していた。道路の位置も幾たびか改まったもので、古道はいつのまにか深い山間に埋もれた。名高い桟も、蔦のかずらを頼みにしたような危い場処ではなくなって、徳川時代の末にはすでに渡ることのできる橋であった。新規に新規にとできた道はだんだん谷の下の方の位置へと降って来た。
道の狭いところには、木を伐って並べ、藤づるでからめ、それで街道の狭いのを補った。長い間にこの木曾路に起こって来た変化は、いくらかずつでも嶮岨な山坂の多いところを歩きよくした。そのかわり、大雨ごとにやって来る河水の氾濫が旅行を困難
