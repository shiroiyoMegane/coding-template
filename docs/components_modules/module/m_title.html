背景タイトル01

  • 使用モジュール
  • 使い方
  • pug
  • html
  • m-bgTitle01

コンテンツの前に要素を入れる必要がある場合、
before:trueにして、 mixin beforeGroup()を宣言することで、 要素の挿入ができる。
コンテンツの後ろ、タイトルの前後についても
以下のオプションをtrueにすることで、挿入可能。

小タイトル、中タイトル、大タイトルについても必要であれば、
smallTitle, title, largeTitleを使用することで挿入可能。
(デフォルトで使用するのは title) 

+m-bgTitle01({
	tag:          "div",                //親タグ名
	baseName:     "m-bgTitle01",   //ベースclass名
	title:        "",                   // タイトル
	smallTitle:   "",                   // 中タイトル
	largeTitle:   "",                   // 大タイトル
	className:    "",                   //追加class名
	before:       false,                //コンテンツの前に入る要素
	after:        false,                //コンテンツの後ろに入る要素
	titleBefore:  false,                //タイトルの前に入る要素
	titleAfter:   false                 //タイトルの後ろに入る要素
})

【シンプル】

+m-bgTitle01({tag:"h3", title:"背景タイトル01"})

【全要素表示】

+m-bgTitle01({tag:"h3", largeTitle:"大タイトル", title:"背景タイトル01（全要素表示）", smallTitle:"少タイトル", className:"is-center",before:true, after:true, titleBefore:true, titleAfter:true})
	mixin beforeGroup()
		// コンテンツの前に入る要素
	mixin titleBefore()
		// タイトルの前に入る要素
	mixin titleAfter()
		// タイトルの後に入る要素
	mixin afterGroup()
		// コンテンツの後に入る要素
		p afterテキスト
.m-bgTitle01
	.group.is-primary
		h3.com-title
			span.is-title tg.title
<div class="m-bgTitle01">
	<div class="group is-primary">
		<h3 class="com-title"><span class="is-title">背景タイトル01</span></h3>
	</div>
</div>
