【margin変更したいとき】
.l-colSet01({col:カラム数, margin:間隔})
カラム数に、並べたい数を入れる。間隔にカラムの間隔を入れる。
カラム数がspのとき必ずしも一つになるとは限らないので、spとpcとで記述を分ける。
他のコンポーネントで、間隔変えたいときは以下のように書いて上書きしてあげるといい
&.is-two
	+MQ_PC(SP_RES_WID01)
		l-colSet01({col:2, margin:PC_MARGIN})
	+MQ_SP(SP_RES_WID01)
		l-colSet01({col:1, margin:SP_MARGIN})
